Abstract

The HPC-FAIR ORCID portion of the ontology provides declarations to semantically describe aspects of an academic or technical publication, such as authorship, affiliations, and information about the publication venue.

HPC Ontology for ORCID: Description back to ToC

The HPC-FAIR ORCID portion of the ontology provides a set of classes to semantically describe publications and other works. Classes, such as hpc:ConferencePaper, can be annotated using with information such as publication year (hpc:publicationYear), volume (hpc:volume), and many other different properties that allow for automatic discovery of properties on publications.
